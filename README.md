## 易经64卦编解码 
### ䷹䷩䷢䷓䷹䷻䷮䷏䷍䷣䷓䷥䷣䷚䷛䷧䷯䷉䷛䷨䷩䷺䷏䷧䷨䷈䷄
### 兌益晉觀兌節困豫大有明夷觀睽明夷頤大過解井履大過損益渙豫解損小畜需

### 原理：
#### Base64刚好对应64卦象，编码时先将目标字符串转成标准的Base64，然后拆分每一个字符，从映射表中取出对应的卦象。
#### 解码同理，先把卦象码通过映射表得到Base64，然后解码Base64。
